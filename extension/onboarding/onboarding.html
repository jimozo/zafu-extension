About Chrome's website access warning

Chrome shows a broad warning because Zafu needs to see crypto-address paste events before a wallet, exchange, dapp, or address-only Telegram Web paste accepts them.

Zafu uses that access for one job: checking pasted crypto addresses before money can be sent to the wrong place, including address-only Telegram Web copy/paste protection.

  • No tabs or activeTab permission
  • No browser history
  • No Telegram chat text, sender identity, groups, message IDs, URLs, mixed messages, or bot commands
  • No advertising analytics; Network Mode is optional
  • No keys, seed phrases, or signing access
